# BiaffineDParser
===========================    
A re-implementation of "[Deep Biaffine Attention for Neural Dependency Parsing](https://arxiv.org/abs/1611.01734)" in [Dynet](https://github.com/clab/dynet)
     
Python: 3.6   

## Run:
python3.6 driver/train.py --config_file config.ctb51.cfg

## Results:
Similar performance as the [pytorch version](https://github.com/zhangmeishan/BiaffineDParser>BiaffineDParser).
